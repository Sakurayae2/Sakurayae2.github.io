排序算法

SakuraGin · 2021-9-9 · 次阅读

排序算法是基础算法之一，常见的有冒泡排序、选择排序、插入排序、快速排序、希尔排序、归并排序、堆排序等。

0.算法总览

1.冒泡排序

1、比较每一对相邻元素的大小，如果后者比前者大，则交换二者；
2、对所有元素重复进行以上比较，直到所有元素都在正确的位置。

1.1.动画演示

1.2.代码实现

function bubbleSort(arr){
    for(var i = 0;i < arr.length - 1;i++){
        for(var j = 0;j <arr.length - 1 - i;j++){
            if(arr[j+1] < arr[j]){
                var temp = arr[j+1];
                arr[j+1] = arr[j];
                arr[j] = temp; 
            }
        }
    }
    return arr;
}

void bubbleSort(int a[],int n){
    for(int i = 0;i < n - 1;i++){
        for(int j = n - 1 - i;j < n;j++){
            if(a[j+1] < a[j]){
                int temp = a[j+1];
                a[j+1] = a[j];
                a[j] = temp;
            }
        }
    }
}

def bubbleSort(arr):
    for i in range(1, len(arr)):
        for j in range(0, len(arr)-i):
            if arr[j] > arr[j+1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

2.选择排序

1、首先在未排序序列中找到最小（大）元素，把它放到排序序列的起始位置；
2、然后再从剩余未排序序列中找到最小（大）元素，放到已排序序列的末尾；
3、重复第二步，直至所有元素排序完成。

2.1.动画演示

2.2.代码实现

function selectionSort(arr) {
    var len = arr.length;
    var minIndex, temp;
    for (var i = 0; i < len - 1; i++) {
        minIndex = i;
        for (var j = i + 1; j < len; j++) {
            if (arr[j] < arr[minIndex]) {     // 寻找最小的数
                minIndex = j;                 // 将最小数的索引保存
            }
        }
        temp = arr[i];
        arr[i] = arr[minIndex];
        arr[minIndex] = temp;
    }
    return arr;
}

3.插入排序

1、将第一待排序序列第一个元素看做一个有序序列，把第二个元素到最后一个元素当成是未排序序列。
2、从头到尾依次扫描未排序序列，将扫描到的每个元素插入有序序列的适当位置。（如果待插入的元素与有序序列中的某个元素相等，则将待插入元素插入到相等元素的后面。）

3.1.动画演示

3.2.代码实现

function insertionSort(arr) {
    var len = arr.length;
    var preIndex, current;
    for (var i = 1; i < len; i++) {
        preIndex = i - 1;
        current = arr[i];
        while(preIndex >= 0 && arr[preIndex] > current) {
            arr[preIndex+1] = arr[preIndex];
            preIndex--;
        }
        arr[preIndex+1] = current;
    }
    return arr;
}
